# **Introduction to Datascience**

## Python

Python is the most widely used language in data and science fields, and one of the most popular languages overall. It is a general-purpose programming language that is easy to learn and has a large community of users. Python has a wide range of libraries and frameworks for data analysis, machine learning, and scientific computing, making it a powerful tool for data scientists. Python was designed to read almost like plain English, making it one of the best first languages to learn.


## Marimo Notebook

Marimo is an interactive notebook for writing and running *python* code. It lets you combine python code, text, and visual outputs in one document, similar to Jupyter notebook. Marimo notebooks are reactive, meaning when you change a variable, all related results update automatically.
They are often used for data analysis, experiments, and creating reproducible computational reports.

## Git Version Control

Git is a tool that tracks changes in your files so you can save versions, go back to earlier work, and collaborate safely on projects. It replaces chaos with a clean, automatic system. I look forward to working with Git in this course to manage my code and collaborate with others.

## Plotly

Plotly is a tool used in python and other programming languages to create interactive charts and graphs that I can zoom, hover over, and explore.

## Polars

Polars is a dataframe library designed for speed and efficiency. It also helps work with large datasets.

## Visual Studio (VS) Code

Its a free code editor made by microsoft built especially for writing code.  VS Code understands your code: it highlights syntax, catches errors as you type, helps you navigate large projects, runs your code, integrates with Git, and can be extended to do almost anything.

## UV Package Management

UV is a modern Python package manager written in Rust that's designed to be fast and reliable.

## Data Science Guide

Data science is the practice of extracting insights and knowledge from data using a combination of statistics, programming, domain expertise, and analytical thinking. It's about turning raw data into actionable information that helps make better decisions.

## Markdown

Markdown is a simple set of symbols you type into plain text to control how it looks when displayed.Markdown is used everywhere: GitHub, Reddit, Notion, Slack, WhatsApp, Discord, Jupyter Notebooks, documentation sites, blogs, and more. Learning it once pays off endlessly.
